Este PR implementa un cliente completo para AWS Cognito siguiendo los patrones de `go-engine`, incluyendo registro de usuarios (`RegisterUser`), autenticación con tokens JWT (`Authenticate`, `ValidateToken`), soporte para MFA (`RespondToMFAChallenge`), renovación de tokens (`RefreshToken`) y recuperación de contraseña (`ForgotPassword`, `ConfirmForgotPassword`). El cliente implementa manejo seguro de secrets mediante variables de entorno (`${VAR_NAME}`), encapsulación del `client_secret` en campo privado, validación robusta de tokens JWT usando JWKS con cache inteligente, resiliencia integrada (circuit breaker y retry), y logging estructurado sin información sensible. La integración con go-engine se realiza mediante configuración opcional en `application.yaml`, agregando el campo `CognitoClient` al `Engine` y el método `GetCognito()` para acceso controlado. Se incluyen 30+ tests unitarios con cobertura del 38.5%, todos pasando sin errores de linting. No hay breaking changes ya que es una adición completamente nueva al framework.
